# Bankore 

This is an banking application which is developed for learning and practising  of fullstack development.

## Description 

Finova is a banking application that allows users to manage accounts, view transactions, and perform basic banking operations.


## Goal

Build a full stack banking application while learning software development.

## Tech Stack

Backend:
Python FastAPI

Database:
MySQL

Frontend:
React

## Features

- Customer management
- Account management
- Transactions
- Authentication

## Learning Goal

Understand:
- Python
- OOP
- Database design
- API development
- Project workflow
